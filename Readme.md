JAVaNC là một ứng dụng web được phát triển với mục tiêu đơn giản hóa việc quản lý bán hàng & thanh toán cho cửa hàng trực tuyến nhỏ.
Hệ thống gồm backend (Java + Spring Boot) và frontend (HTML/CSS/JavaScript) giúp thực hiện các chức năng như:
Quản lý danh sách sản phẩm, cập nhật số lượng tồn kho.
Giỏ hàng & thanh toán: chọn sản phẩm, xác nhận đơn hàng, thay đổi số lượng trong giỏ.
Quản trị đơn hàng: quản lý trạng thái, xem chi tiết, chỉnh giao diện cho các trang About, Shop, Best Sell.
